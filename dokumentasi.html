1. Setup Database (Google Sheets)

Aplikasi ini menggunakan Google Sheets sebagai database gratis. Langkah pertama adalah menduplikasi *template* database ke akun Google Anda.

Langkah-langkah:

  1. Buka link *template* Google Sheets yang telah disediakan.
  2. Klik menu File di pojok kiri atas.
  3. Pilih Make a copy (Buat salinan).
  4. Beri nama file sesuai keinginan (Misal: Database Toko BOS), lalu klik Make a copy.
  5. Pastikan di dalam file tersebut terdapat minimal 3 tab dengan nama persis seperti ini: Produk, Kasir, dan Sheet1.

2. Menghubungkan Server (Google Apps Script)

Agar aplikasi web bisa membaca dan menulis data ke Google Sheets, kita harus mengaktifkan "Jembatan" alias Google Apps Script.

Langkah-langkah Deploy:

  1. Di file Google Sheets yang baru Anda *copy*, klik menu Ekstensi (Extensions) > Apps Script.
  2. Jika kodenya sudah ada, biarkan. Jika kosong, copy-paste kode Code.gs yang sudah Anda miliki.
  3. Klik tombol biru Terapkan (Deploy) di pojok kanan atas, lalu pilih Deployment Baru (New deployment).
  4. Klik ikon roda gigi (Select type) di samping "Pilih jenis", lalu centang Aplikasi Web (Web app).
  5. Pada kolom "Jalankan sebagai (Execute as)", pastikan terpilih Saya (Me).
  6. Pada kolom "Siapa yang memiliki akses (Who has access)", wajib pilih Siapa saja (Anyone).
  7. Klik Terapkan (Deploy). Jika diminta Otorisasi Akses (Authorize Access), klik izinkan dengan akun Google Anda.
  8. Salin URL Aplikasi Web (Web App URL) yang berakhiran /exec. Simpan link ini baik-baik!
⚠️ PENTING: Jika di masa depan Anda mengubah kode di Apps Script, Anda wajib menekan "Terapkan > Kelola Deployment > Ikon Pensil > Versi Baru > Terapkan".

3. Setting File Aplikasi Web

Setelah mendapatkan URL Apps Script, Anda harus menyuntikkannya ke dalam file utama aplikasi kasir.

  1. Buka file script.js di komputer Anda menggunakan Notepad atau Text Editor (VS Code).
  2. Gunakan fitur pencarian (Ctrl + F) dan cari *link* Google Script lama yang berakhiran /exec.
  3. Ganti semua *link* lama tersebut dengan *URL Web App* yang baru saja Anda salin dari Langkah 2. (Total ada 5 tempat yang harus diganti).
  4. Simpan (Save) file script.js tersebut.

4. Manajemen Akun (Tambah User Kasir)

Aplikasi ini memiliki "Gerbang Login". Akun tidak disimpan di kode web, melainkan di Google Sheets agar mudah dikelola oleh Pemilik (Bos).

Cara Menambah/Mengubah Password Kasir:

  1. Buka Google Sheets database Anda.
  2. Masuk ke tab Kasir.
  3. Isi data di baris kosong dengan format:
    • Kolom A (Username): Bebas tanpa spasi (Misal: kasir1)
    • Kolom B (Password): Angka atau huruf (Misal: 12345)
    • Kolom C (Nama_Kasir): Nama asli kasir yang akan muncul di struk (Misal: Budi Santoso).
  4. Data otomatis tersimpan dan bisa langsung digunakan untuk Login.

5. Upload Web ke Cloudflare (Hosting Gratis)

Agar web bisa diakses dari mana saja (HP/Tablet/Laptop) menggunakan *link* resmi tanpa biaya sewa *server*, kita gunakan Cloudflare Pages.

Langkah-langkah Hosting:

  1. Pastikan Anda sudah memiliki 3 file dalam satu folder: index.html, style.css, dan script.js.
  2. Buat akun gratis di dash.cloudflare.com.
  3. Setelah login, klik menu Workers & Pages di menu sebelah kiri.
  4. Pilih tab Pages, lalu klik tombol Upload assets.
  5. Beri nama proyek aplikasi Anda (Misal: kasir-toko-bos), lalu klik Create project.
  6. Tarik dan lepas (Drag & Drop) folder kasir-main yang berisi 1 folder dan 3 file tadi ke dalam kotak yang disediakan Cloudflare.
  7. Tunggu proses *upload* selesai, lalu klik Deploy site.
  8. Selamat! Anda akan mendapatkan *link* web gratis (misal: kasir-toko-bos.pages.dev) yang siap dipakai berjualan.

6. Penjelasan Fitur Aplikasi Lengkap

Berikut adalah cara menggunakan fitur-fitur yang tersedia di halaman kasir:

Fitur Penjelasan & Cara Penggunaan
Scanner Kamera 📷 Klik ikon kamera di sebelah kotak pencarian. Kamera HP/Laptop akan menyala. Arahkan barcode barang, maka item akan otomatis masuk ke keranjang.
Menu Favorit ⭐ Jalan pintas untuk barang paling laris atau tanpa barcode (es batu, kresek). Untuk menambahkannya, ketik kata YA di kolom F (Favorit) pada tab "Produk" di Google Sheets.
Diskon Per Item 🏷️ Berfungsi memotong harga pada satu barang spesifik. Klik barang di keranjang, lalu klik "DISC ITEM" dan masukkan nominal diskon (Rupiah).
Override Harga 💲 Berfungsi untuk mengubah harga jual secara paksa (misal untuk harga grosir). Klik barang di keranjang, pilih tombol "HARGA", dan ketik harga baru.
Simpan Draft 📄 Gunakan tombol "NEW (F11)" untuk menahan transaksi pelanggan yang belum selesai (misal: dompet tertinggal). Panggil kembali transaksinya dengan tombol "OPEN DRAFT".
Dashboard Laporan 📊 Klik tombol Laporan untuk melihat omzet Hari Ini, 7 Hari Terakhir, dan Bulan Ini secara *Real-Time*. Termasuk tabel 50 transaksi terakhir.
Download / Print PDF 📄 Terdapat pada Modal Struk dan Modal Laporan. Berfungsi mencetak dokumen sangat rapi (hanya hitam putih tanpa elemen web) untuk di-save sebagai PDF atau diprint ke printer A4 biasa.
Cetak Thermal 🖨️ Setelah transaksi (Bayar/F12), klik tombol "Cetak Thermal". Web akan terhubung langsung via Bluetooth/Kabel serial ke Printer Kasir (Web Serial API).
